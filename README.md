# TTC_Subway_Analysis
An analysis of TTC delay data for estimation of downtime and optimal commute routing.
## Motivation
https://vt.tiktok.com/ZSPXRLtPS/ 
## Goal
Determine the delay situations which warrant walking instead of waiting. Estimate odds 
of shuttle bus.

### Questions to Answer
Does the number of delays due to patrons increase in relation to season.
Does the number of delays due to meechanical issues increase in relation to season

## Data
TTC delay data: https://open.toronto.ca/dataset/ttc-subway-delay-data/
